
INTERNET.PY
================

    Uma simples calculadora de todos os sinais


----

Executar
----------

Efetuar a execução do código-fonte:


          $ python3 internet.py


----

Uso 
---
:

          Fazer calculos que chegam a sua velocidade de internet


----

Exemplos
--------
           tamanho do arquivo em gigabytes:100
           sua velocidade de internet:30
           3 minutos -06
